How to Fill Streamer Fields

Screen ID

Use the TwitCasting ID that appears after twitcasting.tv/ in the channel URL.

Interval

How often the recorder checks this streamer. Example: 5 means every 5 seconds.

Target Folder

Recordings for this streamer are saved under the Recordings/ subfolder you choose here.

Live Password

Only fill this if the stream uses a TwitCasting secret word.
